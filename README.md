 Traffic Accident Risk Prediction & Road Safety Analytics

 Overview
This project analyzes real-world traffic accident data to identify high-risk
patterns and predict accident severity. The goal is to support data-driven
decision-making for road safety and urban planning.

 Problem Statement
Traffic accidents cause significant loss of life and economic damage.
Authorities need analytical tools to:
- Identify accident hotspots
- Understand factors contributing to severe injuries
- Improve road safety policies

 Data Source
The project uses open traffic accident data provided by government and public
sources.

Skills & Tools Used
- Python
- Pandas & NumPy (data analysis)
- SQL (data querying)
- Statistics (hypothesis testing)
- Machine Learning (classification models)
- Git & GitHub (version control)
